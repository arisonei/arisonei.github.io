# [解读沉默：当言语成为了冲突的导火索](https://arisonei.github.io/)

有一种现象在情侣关系中非常普遍：当你想表达自己的不满时，往往会被对方的“沉默”打断。你说：“我觉得你今天太忙了，都没时间陪我。”她没有反驳，只是低头玩手机。你愈发激动，甚至开始指责她不关心你。就在这时，她站起身来，声音变得很冷：“那你算了。”

这种看似“无理取闹”的争吵，实际上是深层需求的冲突。你渴望的是“陪伴”，而她的沉默可能是“压力”。她选择沉默，是因为她不想在压力面前表现出脆弱；而你的言语，却恰恰触碰到了她最敏感的神经。

这种情况下，传统的“你说我听”式沟通往往无效。因为双方的注意力都被情绪占据了。我们需要转变方式，采用“非暴力沟通”的模式。

首先，尝试观察而不是评判。当她沉默时，不要立刻贴标签“你不爱我”，而是观察她的行为：“我看到你今天没有像往常一样主动给我发信息”。其次，表达感受而不是指责：“我感觉有点被冷落”。最后，表达需求：“我需要在忙碌的日子里感受到你的存在”。

通过这种方式，你们可以将“沉默”和“言语”重新定义为沟通的桥梁，而不是冲突的导火索。当我们停止指责，转而表达内心的真实需求时，沉默不再是冷漠，而是需要被倾听的求救信号。
